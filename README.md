# Student Training Management System

A frontend web application built using **HTML, CSS, and JavaScript** to manage student training and placement status.

## Features
- User registration with form validation
- Login authentication using localStorage
- Interactive dashboard to manage students
- Add, edit, delete, and track placement status
- Filter and sort student records

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Purpose
This project was developed as part of my learning journey to strengthen my understanding of JavaScript logic, DOM manipulation, and client-side data handling.

## Note
This project uses **localStorage** for data persistence and is intended for learning purposes only.
